# Synthesis Framework

- Company Name
- Size (number of employees/total annual revenue)
- Geographic Footprint

- Primary Client Types

- Delivery Orientation

## Organizational Model
Overview of business capabilities aligned to functions and services (diagram).

## Functional Decomposition

- Purpose
- Key Activities
- Inputs
- Outputs
- Roles
- Interfaces

Key business processes with high-level steps.

## Interaction Model
Overview of the process/workflows (diagram) with a description.

## Variability and Assumptions
If completed by an agent, all completed items should be tagged as indicated. Observed items should include a reference where possible.

- **Observed:** The listed item was identified through source material.
- **Inferred:** The listed item was derived from supporting evidence.
- **Uncertain:** The listed item is logical to the identified topic, but clear evidence is unavailable.
